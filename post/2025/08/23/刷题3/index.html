Saturday, August 23, 2025 1 minute read

今天做了三个笔试，记录下相关考点

1. case的特殊用法

always @(*)begin
case(1'b1)
    s1:state 1 ;
    s2:state 2 ; 
    s3:state 3 ; 
    default：state 4 ;
endcase

优先级编码 用的比较少，等同于

if (s1)
    state 1;
else if (s2)
    state 2;
else if (s3)
    state 3;
else 
    state 4;

回顾下case、casez、casex

case(expression)
item : statement;

当 item 和 expresion 完全匹配才会执行

casez(expression)
item : statement;

匹配时不关心Z值

casex(expression)
item : statement;

匹配时不关心 x/z 的值，最好不要使用casex

2. 贴片电阻封装和功率

  • 封装 功率
  • 0201 1/20W
  • 0402 1/16W
  • 0603 1/10W
  • 0805 1/8W
  • 1206 1/4W

3. 信号完整性

电源完整性和信号完整性的相互影响

常见阻抗标准：单端50Ω、单端75Ω、差分100Ω、90Ω。

高速信号使用介电常数低，介电损耗正切角的板材。

串联端接电路靠近发送端。

峰值电流控制降压电路中次谐波震荡和谐波补偿产生边界点在于50%占空比。

4. 其他

106对应10和6，10代表数值、6代表后面0的个数，所以106Ω为10,000,000Ω。

5. 数电

通用逻辑门：NAND、NOR

  • 单独NAND
  • 单独NOR
  • 与、或、非门组合

可以实现任意逻辑；

6 电路分析

always @(posedge clk or negedge rst_n)begin
    if(!rst_n)begin
        a <= 1'b0;
    end
    else if (b>1'b0)
        a <= b;
end

b为高电平时，a才会被赋值为b -> 电平敏感；
